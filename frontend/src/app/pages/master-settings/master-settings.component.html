{{ getTabLabel(activeTab) }} Module Definition

Customer Master: Manages non-patient, corporate, and contract accounts (e.g., drug trials, wellness companies, and corporate packages).

This module configures client details, custom packaging rules, discount agreements, and corporate invoicing details. Enabling this streamlines wholesale ledger transactions and company payouts.

Administrator Only

Area Master: Defines geographical sectors, pickup zones, and delivery routes.

This setting maps collection boys to clinic networks and patient locations. It enables geo-performance analysis, optimizing collection schedules, and improving turnaround time for home collection visits.

Administrator Only

Contacts Directory: A central laboratory phonebook for support contacts, suppliers, and emergency hotlines.

Tracks diagnostic analyzer support details, vendor details, and repair engineers to speed up callout logs when hardware or software issues occur.

Administrator Only

Result Note Template: Save clinical disclaimers, text footnotes, and result interpretations.

Standardizes reports and saves pathologists time by inserting complex interpretations (e.g., standard normal values or risk factors) into patient reports in one click.

Set Result Template: Configure document layouts and paragraph grids for narrative tests.

Used for pathology, biopsy readings, radiology scans, and ultrasound records where results cannot be expressed as basic numeric values.

Set Hospital Collection: Set up collection center mapping and payouts.

Tracks deposits, logs collection boys' handovers, and manages commission splits/settlements automatically with referral centers.
